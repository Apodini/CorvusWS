ExtendedNestedReadAllAuthModifier

public final class ExtendedNestedReadAllAuthModifier<
    A: ExtendedAuthEndpoint,
    I: CorvusModel,
    U: CorvusModelAuthenticatable>:
ExtendedReadEndpoint

A class that wraps a component which utilizes an .auth() modifier. Differs from ExtendedAuthModifier by authenticating on the user of an intermediate parent I of A.QuerySubject. Requires an object U that represents the user to authorize.
