ExtendedResponseModifier

public final class ExtendedResponseModifier<E: ExtendedRestEndpoint, R: CorvusResponse>:
ExtendedRestEndpoint where E.Element == R.Item

A class that wraps a component which utilizes a .respond(with:) modifier. That allows Corvus to chain modifiers, as it gets treated as any other struct conforming to ExtendedRestEndpoint.
