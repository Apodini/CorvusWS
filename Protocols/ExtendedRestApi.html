ExtendedRestApi

public protocol ExtendedRestApi : ExtendedEndpoint, RouteCollection

A component that is used at the root of each API and can get registered to a Vapor Application by conforming to RouteCollection. It also conforms to ExtendedEndpoint to remain composable itself.
