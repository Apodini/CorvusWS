ExtendedRestModifier

public class ExtendedRestModifier<R> : ExtendedRestEndpoint where R : ExtendedRestEndpoint

A class that wraps an ExtendedRestEndpoint with additional functionalty. This allows Corvus to chain modifiers, as it gets treated as any other struct conforming to ExtendedRestEndpoint.
