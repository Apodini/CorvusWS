ExtendedAuthModifier

public class ExtendedAuthModifier<
    A: ExtendedAuthEndpoint,
    U: CorvusModelAuthenticatable>:
ExtendedRestModifier<A>, ExtendedAuthEndpoint

A class that wraps a component which utilizes an .auth() modifier. That allows Corvus to chain modifiers, as it gets treated as any other struct conforming to ExtendedAuthEndpoint. Requires an object U that represents the user to authorize.
