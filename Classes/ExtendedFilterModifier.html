ExtendedFilterModifier

public final class ExtendedFilterModifier<
    R: ExtendedReadEndpoint
>: ExtendedReadEndpoint

A class that wraps a component which utilizes a .filter() modifier. That allows Corvus to chain modifiers, as it gets treated as any other struct conforming to ExtendedReadEndpoint.
