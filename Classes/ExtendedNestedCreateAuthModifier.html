ExtendedNestedCreateAuthModifier

public final class ExtendedNestedCreateAuthModifier<
    A: ExtendedCreateEndpoint,
    I: CorvusModel,
    U: CorvusModelAuthenticatable>:
ExtendedNestedAuthModifier<A, I, U>, ExtendedCreateEndpoint

A class that wraps an ExtendedCreate component which utilizes an .auth() modifier. That allows Corvus to chain modifiers, as it gets treated as any other struct conforming to ExtendedNestedCreateAuthModifier. Requires an object U that represents the user to authorize.
